# Risotto-boilerplate

First draft of a Risotto-boilerplate

===========
### Install

Clone the repository or use the Risotto "--scaffolding" flag.

Execute inside the root folder after cloning/scaffolding:
```bash
$  npm install 				# installs all nodejs requirements
```
